# 🎯 Template Usage (For Others)

Want to use this as your portfolio?

### 1. Clone the repo

```
git clone https://github.com/Varun-Dadlani/digital-portfolio.git
```

### 2. Open template

Go to:

```
template/index.html
```

### 3. Customize

Replace:

* `Your Name`
* `your-profile-image.jpg`
* Project details
* Social links

### 4. Fix paths (if needed)

If you move files, update:

* CSS path
* JS path
* Assets path
